# 炼金术量化交易系统 V2.1

[![Python](https://img.shields.io/badge/Python-3.14-blue)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**技术面动量选股 + 基本面价值投资双模式**

炼金术量化交易系统是一套基于Python开发的智能A股量化交易系统，采用"炼金术"命名，寓意通过科学的量化方法，从海量股票中提炼出具备投资价值的优质标的。

## ✨ 核心特点

- **双模式选股**：技术面动量 + 基本面价值双重筛选
- **智能筛选**：全盘扫描并建立基础池股票，进行多维度技术分析
- **金字塔仓位**：动态仓位管理，风险可控
- **行业分散**：跨行业配置，降低集中度风险
- **回测验证**：基于Backtrader引擎的历史回测
- **专业报告**：自动生成PDF投资分析报告

## 📋 功能概览

| 功能模块 | 描述 |
|---------|------|
| 技术面选股 | MACD、多均线、BOLL等多指标筛选 |
| 基本面选股 | PE、PB、ROE、增长率等财务指标 |
| 组合策略 | 技术+基本面双验证，提高胜率 |
| 回测引擎 | 基于Backtrader框架的历史数据回测 |
| 风险管理 | 金字塔加仓、分批止盈策略 |

## 🔍 选股策略

### 3.1 技术面动量选股

系统采用多指标共振策略，要求股票同时满足：

- **多均线多头排列** - 5日>10日>30日均线，确保中期趋势向上
- **MACD正向金叉** - DIF>0且DEA>0，表明动量强劲
- **突破BOLL中轨** - 当前价格>BOLL中轨(20日均线)，打开上升通道
- **3日累计上涨** - 最近3日累计涨幅为正，确认短期动能

### 3.2 基本面价值选股

基本面选股采用价值投资理念：

- **市值筛选** - 总市值>100亿元，确保流动性
- **估值筛选** - 0<市盈率(动态)<30，避免估值过高和亏损企业
- **行业分散** - 跨行业配置(金融、医药、消费、能源、制造、公用)
- **质量筛选** - 排除ST股票，换手率>0.5%保证流动性

## 📊 技术指标

| 指标 | 计算公式 | 应用说明 |
|-----|---------|---------|
| MA均线 | N日收盘价平均值 | 判断趋势方向 |
| MACD | DIF=EMA12-EMA26 | 判断动量强弱 |
| BOLL | 中轨±2倍标准差 | 判断波动通道 |
| RSI | 相对强弱指标 | 判断超买超卖 |
| KDJ | 随机指标 | 判断买卖时机 |

## 🚀 快速开始

### 环境要求

- Python 3.14+
- 依赖包：akshare, backtrader, pandas, reportlab, openpyxl

### 安装

```bash
# 克隆仓库
git clone https://github.com/s3805/Alchemy-Quant.git
cd Alchemy-Quant

# 安装依赖
pip install -r requirements.txt
```

### 使用指南

#### 步骤一：运行选股程序

```bash
# 技术面动量选股(默认选20只)
python alchemy_system.py --mode momentum --max-stocks 20

# 基本面价值选股
python alchemy_system.py --mode value

# 组合模式(两者结合)
python alchemy_system.py --mode both
```

#### 步骤二：查看选股结果

系统自动生成Excel文件：`alchemy_momentum_stocks.xlsx`，包含股票代码、名称、技术指标等详细信息，按MACD-DIF强度排序。

文件保存位置：`/Users/xugang/Desktop/炼金术量化交易系统/`

#### 步骤三：生成投资分析报告

```bash
python generate_alchemy_report.py
```

自动为每只股票生成PDF分析报告，包含基本面、技术面、投资建议等。

#### 步骤四：查看回测结果

系统自动运行回测，输出收益率、最大回撤等指标和交易信号明细。

## ⚠️ 风险提示

**重要风险提示**

1. **历史表现不代表未来收益**
   - 系统基于历史数据回测，市场环境不断变化
   - 过去业绩不预示未来表现

2. **技术分析局限性**
   - 技术指标存在滞后性
   - 突发事件无法预测
   - 需结合基本面分析

3. **资金管理建议**
   - 单只股票仓位不超过20%
   - 设置止损位(建议-8%)
   - 分批建仓，分批止盈
   - 保留现金储备

4. **系统性风险**
   - 市场整体下跌风险
   - 政策变化风险
   - 黑天鹅事件风险

5. **使用建议**
   - 本系统仅供参考，不构成投资建议
   - 请结合自身风险承受能力
   - 建议咨询专业投资顾问
   - **股市有风险，投资需谨慎**
